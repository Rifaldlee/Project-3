FAQ'S

What does the i on the Intel processor mean?

The i writing on the Intel cpu is the series marker of the processor

What is the number after the letter on Intel processors?

The number after the letter on the Intel cpu is a Generation marker, for example (i7) it means the 7th generation series i processor

What is VGA and its function?

VGA (Video Graphics Adapter) is a computer hardware that functions to translate the display to the monitor screen. VGA is also often referred to as a graphics card.

About the motherboard and what does it contain?

Basically, the motherboard is a circuit board that is inside a computer case to which all the components of the computer are connected. The motherboard provides connectivity between the computer hardware components, such as the processor (CPU), memory (RAM), hard drive, and video card.

How do power supplies work?

Basically Power Supply requires a power source which is then converted into energy that drives electronic devices. The working system is quite simple, namely by converting 120V power into a flow form with high power according to the requirements of these components.

What is storage on a computer and how many types are there?

Storage is storage, storage, media used to store data that is processed by a computer. There are two types of storage devices that are used as secondary storage on computers: HDDs and SSDs. HDDs are the more traditional of the two types, while SSDs are quickly overtaking HDDs as the technology of choice for secondary storage.

What is computer RAM?

RAM is a very vital component. The function of RAM itself processes incoming data in various ways into memory. The RAM function will work optimally depending on the capacity and speed of data access. The bigger the RAM installed, the faster the computer works.

What is the function of the computer's CPU cooler?

CPU Cooler is one component on a computer that can be said to be a fairly vital tool, considering that the CPU cooler is a tool that functions to dissipate heat so that the temperature on the processor is maintained and has no effect. to a decrease in performance caused by an increase in temperature.
